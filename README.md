# DSA From Zero

A personal repository documenting my journey of learning Data Structures and Algorithms from the fundamentals.

## Overview

This repository contains C++ implementations, practice problems, and notes covering core DSA concepts. The goal is to build a strong understanding of problem-solving techniques and data structures through hands-on coding.

## Topics Covered

* Arrays
* 2D Arrays
* Strings & Character Arrays
* Linked Lists
* Queues
* Binary Trees
* Binary Search Trees (BST)
* Recursion
* Backtracking
* Sorting Algorithms
* Greedy Algorithms
* Divide and Conquer
* Bit Manipulation
* Object-Oriented Programming (OOP)
* STL Basics

## Purpose

* Strengthen DSA fundamentals
* Practice implementation from scratch
* Improve problem-solving skills
* Prepare for coding interviews and competitive programming

## Tech Stack

* C++
* STL
* GCC / G++

## Repository Structure

Each folder/file focuses on a specific topic and contains implementations, practice problems, and concept-based exercises.
